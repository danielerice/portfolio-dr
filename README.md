# Daniel Rice — Portfolio

Personal portfolio site with a retro sci-fi, pixel-art aesthetic. Built with React, Vite, and PixiJS.

## Stack

- **React 19** — UI
- **Vite 8** — dev server and bundler
- **PixiJS 8** — WebGL animation layer (background, splash effects)
- **React Router 7** — routing
- **Formspree** — contact form
- **Vitest + Testing Library** — tests

## Development

Requires Node 24 (see `.nvmrc` — run `nvm use`).

```bash
npm install
npm start        # dev server at http://localhost:3000
npm run build    # production build to build/
npm run preview  # serve the production build locally
npm test         # run tests
```
